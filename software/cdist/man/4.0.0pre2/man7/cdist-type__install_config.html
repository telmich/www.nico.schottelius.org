1. NAME

cdist-type__install_config - run cdist config as part of the installation

2. DESCRIPTION

This cdist type allows you to run cdist config as part of the installation. It does this by using a custom _remote{copy,exec} prefix which runs cdist config against the /target chroot on the remote host.

3. REQUIRED PARAMETERS

None.

4. OPTIONAL PARAMETERS

chroot
where to chroot before running grub-install. Defaults to /target.

5. EXAMPLES

__install_config

__install_config --chroot /mnt/somewhere
